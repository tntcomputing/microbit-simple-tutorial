# microbit-simple-tutorial

A minimal working MakeCode micro:bit tutorial test.

## Use as Extension

This repository can be added as an **extension** in MakeCode.

* open [https://makecode.microbit.org/](https://makecode.microbit.org/)
* click on **New Project**
* click on **Extensions** under the gearwheel menu
* search for **https://github.com/tntcomputing/microbit-simple-tutorial** and import

## Tutorial

After importing this extension, the tutorial will be available in your project.
