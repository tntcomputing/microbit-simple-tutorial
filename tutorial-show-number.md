# Show Number Tutorial

```template
basic.forever(function () {
	
})
```

## Introduction @showdialog

Welcome to your first MakeCode micro:bit tutorial!

In this tutorial, you will learn how to display a number on the micro:bit LED screen.

## Step 1

Add a ``||basic:show number||`` block inside the ``||basic:on start||`` block.

```blocks
basic.showNumber(0)
```

## Step 2

Change the number from **0** to **42**.

```blocks
basic.showNumber(42)
```

## Step 3

Great job! Click **Done** to see your code. You should see the number 42 displayed on the micro:bit simulator!
