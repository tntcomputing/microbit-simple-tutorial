# Simple Tutorial

## Step 1 @showdialog

Welcome to your first MakeCode micro:bit tutorial!

## Step 2

Add a ``||basic:show number||`` block inside the ``||basic:on start||`` block and set the number to **42**.

```blocks
basic.showNumber(42)
```

## Step 3

Click the **Play** button on the simulator to test your code. You should see the number 42 displayed on the micro:bit screen!
